# EcoSort: Your Intelligent Waste Assistant ♻️
> An AI-powered web application helping households segregate waste correctly to save the planet.

## Features
*   **Instant AI Analysis:** Upload an image and get a classification in seconds.
*   **Actionable Advice:** Tells you exactly how to dispose of the item (Recycle, Compost, Landfill, etc.).
*   **Mobile Friendly:** Works seamlessly on desktop and mobile browsers.
*   **Privacy Focused:** Images are processed in the cloud and not permanently stored.

## Tech Stack
*   **Language:** Python 3.9+
*   **Deep Learning:** TensorFlow & Keras (MobileNetV2)
*   **Frontend:** Streamlit
*   **Deployment:** Streamlit Cloud

## How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
